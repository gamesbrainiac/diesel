default_process_commit_error

Default implementation of CommitErrorProcessor::process_commit_error(), used for MySql and Sqlite connections. Returns CommitErrorOutcome::RollbackAndThrow if the transaction depth is greater than 1, the error is a DatabaseError and the error kind is either DatabaseErrorKind::SerializationFailure or DatabaseErrorKind::ReadOnlyTransaction
